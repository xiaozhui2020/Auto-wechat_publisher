# 公众号文章自动制作与发布系统

1. 以markdown的格式储存每一篇文章
2. 基于获取到的数据，生成公众号文章
   1. 标题
   2. 作者
   3. 文章内容
      1. 二维码商品推广【秀米】
      2. 温馨提示【mdnice】
      3. 共享试题
         1. 年份标题
         2. 版本1+内置超链接
         3. 微点评
         4. 版本2+内置超链接
         5. 微点评
      4. 共享网盘
         1. 内容介绍
         2. 网盘链接
      5. 投票
   4. 封面
   5. 摘要
   6. 原创
   7. 赞赏
   8. 合集
3. 上传图片并获取图片超链接
4. 草稿箱内容创作
5. 定时批量发布
6. 基于mdnice制作的模板公众号文章样式
7. 基于秀米制作的模板公众号文章样式
